# Euromap77
Contains the models and classes for Euromap 77.

# What is Euromap 77
Euromap 77 is about data interchange between Injection Moulding Machines (IMM) and Manufacturing Engineering Systems (MES).

# Used Technology
Euromap 77 is based on OPC/UA.
